# ErmJS

ErmAI JavaScript SDK for browsers.

## Classes

- `ChatGPSimon(apiKey, chat)` - Groq-powered AI with memory
- `Lumina(apiKey, instruction, chat)` - Lightweight assistant with memory
- `Translate(apiKey)` - Translation module
- `Billy(apiKey, modelId)` - Google Generative AI with localStorage memory

## Build

```bash
npm install
npm run build
```
